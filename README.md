### Rose Rocket Driver Challenge

## How to Run the App

1. Fork and clone the repository.
2. In terminal, run the command 'npm install'
3. Run 'node server.js'
4. In another terminal window, run 'npm start'

##Dependancies

- express
- body-parser
- react
- react-dom
- react-scripts
- konva
- react-konva
- node-sass
- npm run-all
