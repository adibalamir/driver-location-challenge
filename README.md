# Rose Rocket Driver Challenge

## How to Run the App

1. Copy or fork and clone the repository.
2. In terminal, run the command 'npm install'
3. Run 'node server.js'
4. In another terminal window, run 'npm start'

## Dependancies

- Node v10.15.0

### Back-end

- express
- body-parser
- npm-run-all

### Front-end

- react
- react-dom
- react-scripts
- konva
- react-konva
- node-sass
